# Форматированное приветствие

Напишите функцию `FormatGreeting`, которая принимает имя (`string`) и возраст (`int`) и возвращает строку вида `"Привет, Иван! Тебе 25 лет."`.

Используйте `fmt.Sprintf` для форматирования.

## Пример

| Имя       | Возраст | Выход                          |
|-----------|---------|--------------------------------|
| `"Иван"`  | `25`    | `"Привет, Иван! Тебе 25 лет."` |
| `"Анна"`  | `1`     | `"Привет, Анна! Тебе 1 лет."`  |
| `""`      | `0`     | `"Привет, ! Тебе 0 лет."`      |
