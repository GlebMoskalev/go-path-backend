# Уровень температуры

Напишите функцию `TempLevel`, которая принимает температуру в градусах Цельсия (`float64`) и возвращает текстовую метку:

- ниже 0 → `"мороз"`
- 0..14 → `"холодно"`
- 15..24 → `"комфортно"`
- 25..34 → `"жарко"`
- 35 и выше → `"опасная жара"`

Используйте именованные константы для границ диапазонов.

## Пример

| Вход    | Выход             |
|---------|-------------------|
| `-10.0` | `"мороз"`         |
| `0.0`   | `"холодно"`       |
| `20.0`  | `"комфортно"`     |
| `30.0`  | `"жарко"`         |
| `40.0`  | `"опасная жара"`  |
