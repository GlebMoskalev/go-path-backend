# Форматирование таблицы

Напишите функцию `TableRow`, которая принимает название (`string`) и число очков (`int`) и возвращает строку вида:

```
"Ivan               |  42"
```

Правила форматирования:
- Название выравнивается **по левому краю**, занимает ровно **20 символов**
- После `|` число выравнивается **по правому краю**, занимает ровно **4 символа**

Подсказка: `%-20s` в `fmt.Sprintf` выравнивает строку по левому краю в поле шириной 20.

## Пример

| Название       | Очки | Выход                           |
|----------------|------|---------------------------------|
| `"Ivan"`       | `42` | `"Ivan               |  42"`    |
| `"Alex Ivanov"`| `100`| `"Alex Ivanov        | 100"`    |
| `""`           | `0`  | `"                    |   0"`   |
