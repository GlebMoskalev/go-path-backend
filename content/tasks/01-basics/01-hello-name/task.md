# Приветствие по имени

Напишите функцию `Hello`, которая принимает имя и возвращает строку приветствия.

## Пример
| Вход      | Выход              |
|-----------|--------------------|
| `"Глеб"`  | `"Привет, Глеб!"`  |
| `"World"` | `"Привет, World!"` |
| `""`      | `"Привет, !"`      |
