# Валидация возраста

Напишите функцию `ValidateAge`, которая принимает возраст `int` и возвращает `error`.

Правила:
- Если возраст < 0 → `errors.New("возраст не может быть отрицательным")`
- Если возраст > 150 → `errors.New("возраст не может превышать 150 лет")`
- В остальных случаях → `nil`

## Пример

| Вход  | Ошибка                                      |
|-------|---------------------------------------------|
| `25`  | `nil`                                       |
| `-1`  | `"возраст не может быть отрицательным"`     |
| `200` | `"возраст не может превышать 150 лет"`      |
| `0`   | `nil`                                       |
| `150` | `nil`                                       |
