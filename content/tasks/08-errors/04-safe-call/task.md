# Безопасный вызов

Напишите функцию `SafeCall`, которая принимает функцию `func()` и выполняет её безопасно: если функция паникует — перехватить панику и вернуть её как `error`.

- Если функция завершилась нормально → `nil`
- Если функция вызвала `panic(msg)` → `fmt.Errorf("panic: %v", msg)`

`recover()` работает только внутри `defer` — это ключевое ограничение.

## Пример

```go
SafeCall(func() {
    // нормальное выполнение
})
// → nil

SafeCall(func() {
    panic("что-то пошло не так")
})
// → error: "panic: что-то пошло не так"
```

| Функция                          | Результат                       |
|----------------------------------|---------------------------------|
| нормальная функция               | `nil`                           |
| `panic("ошибка")`                | `error("panic: ошибка")`        |
| `panic(42)`                      | `error("panic: 42")`            |
