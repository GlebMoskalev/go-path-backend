# Дженерик Map

Напишите обобщённую функцию `MapFn`, которая принимает слайс `[]T` и функцию преобразования `func(T) U`, и возвращает новый слайс `[]U`, где каждый элемент — результат применения функции.

Используйте два типовых параметра: `[T any, U any]`.

## Пример

```go
MapFn([]int{1, 2, 3}, func(n int) string {
    return strconv.Itoa(n)
})
// ["1", "2", "3"]

MapFn([]string{"hello", "world"}, func(s string) int {
    return len(s)
})
// [5, 5]
```

| Слайс             | Функция           | Выход             |
|-------------------|-------------------|-------------------|
| `[1, 2, 3]`       | `n * n`           | `[1, 4, 9]`       |
| `["hi", "bye"]`   | `len(s)`          | `[2, 3]`          |
| `[]`              | любая             | `[]`              |
