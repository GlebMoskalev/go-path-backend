# Дженерик Filter

Напишите обобщённую функцию `Filter`, которая принимает слайс любого типа `[]T` и функцию-предикат `func(T) bool`, и возвращает новый слайс, содержащий только элементы, для которых предикат вернул `true`.

Используйте типовой параметр `[T any]` (Go 1.18+).

## Пример

```go
Filter([]int{1, 2, 3, 4, 5}, func(n int) bool { return n%2 == 0 })
// [2, 4]

Filter([]string{"go", "rust", "java"}, func(s string) bool { return len(s) == 2 })
// ["go"]
```

| Слайс                      | Предикат             | Выход           |
|----------------------------|----------------------|-----------------|
| `[1, 2, 3, 4, 5]`          | `n % 2 == 0`         | `[2, 4]`        |
| `["go", "rust", "java"]`   | `len(s) == 2`        | `["go"]`        |
| `[]`                       | любой                | `[]`            |
