# Площадь фигуры

Реализуйте интерфейс `Shape` с методом `Area() float64` для двух структур:

- `Rectangle` с полями `Width`, `Height float64`
- `Circle` с полем `Radius float64`

Также напишите функцию `TotalArea`, которая принимает слайс `[]Shape` и возвращает сумму площадей.

Используйте `math.Pi` для вычисления площади круга.

## Пример

| Фигура                       | Area      |
|------------------------------|-----------|
| `Rectangle{3, 4}`           | `12`      |
| `Circle{5}`                 | `78.5398` |
| `TotalArea(rect, circle)`   | `90.5398` |
