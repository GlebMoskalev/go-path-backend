# Подсчёт простых чисел

Напишите функцию `CountPrimes`, которая принимает число `n` и возвращает количество простых чисел в диапазоне от 2 до `n` включительно.

Число считается простым, если оно делится только на 1 и на само себя.

## Пример

| Вход | Выход | Простые числа               |
|------|-------|-----------------------------|
| `10` | `4`   | 2, 3, 5, 7                  |
| `1`  | `0`   | (нет простых до 1)          |
| `2`  | `1`   | 2                           |
| `20` | `8`   | 2, 3, 5, 7, 11, 13, 17, 19 |
