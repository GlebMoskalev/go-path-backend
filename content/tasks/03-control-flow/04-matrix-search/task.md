# Поиск в матрице

Напишите функцию `FindInMatrix`, которая принимает двумерный слайс `[][]int` и значение `target int`, и возвращает индексы `(row, col)` первого вхождения.

- Если значение найдено — вернуть `(row, col, true)`
- Если не найдено — вернуть `(0, 0, false)`

Для выхода из двух вложенных циклов сразу используйте именованный `break` с меткой.

## Пример

```
matrix := [][]int{
    {1, 2, 3},
    {4, 5, 6},
    {7, 8, 9},
}
```

| Цель | Строка | Столбец | Найдено |
|------|--------|---------|---------|
| `5`  | `1`    | `1`     | `true`  |
| `1`  | `0`    | `0`     | `true`  |
| `9`  | `2`    | `2`     | `true`  |
| `42` | `0`    | `0`     | `false` |
