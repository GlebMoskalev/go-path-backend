# Run-length кодирование

Напишите функцию `RunLength`, которая принимает строку и возвращает её сжатую версию методом run-length encoding.

Правило: каждая серия одинаковых символов заменяется количеством повторений и символом. Если символ встречается один раз — число 1 не пишется.

## Пример

| Вход        | Выход      |
|-------------|------------|
| `"aaabbc"`  | `"3a2bc"`  |
| `"aabbcc"`  | `"2a2b2c"` |
| `"abc"`     | `"abc"`    |
| `""`        | `""`       |
| `"aaaa"`    | `"4a"`     |

Подсказка: итерируйтесь по строке с помощью `for range`, отслеживайте текущий символ и счётчик его повторений.
