# Оценка по баллам

Напишите функцию `Grade`, которая принимает количество баллов (`int`, от 0 до 100) и возвращает буквенную оценку:

- 90–100 → `"A"`
- 80–89  → `"B"`
- 70–79  → `"C"`
- 60–69  → `"D"`
- 0–59   → `"F"`

Используйте `switch` без выражения (switch с условиями в case).

## Пример

| Вход | Выход |
|------|-------|
| `95` | `"A"` |
| `85` | `"B"` |
| `75` | `"C"` |
| `65` | `"D"` |
| `45` | `"F"` |
| `0`  | `"F"` |
