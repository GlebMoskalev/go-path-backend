# Пересечение слайсов

Напишите функцию `Intersect`, которая принимает два слайса `[]int` и возвращает новый слайс, содержащий элементы, присутствующие **в обоих** слайсах. Дубликаты в результате не допускаются.

- Порядок элементов в результате должен соответствовать порядку их первого появления в слайсе `a`
- Используйте map для эффективного поиска

## Пример

| a                | b              | Выход         |
|------------------|----------------|---------------|
| `[1, 2, 3, 4]`   | `[3, 4, 5, 6]` | `[3, 4]`      |
| `[1, 2, 2, 3]`   | `[2, 3, 3]`    | `[2, 3]`      |
| `[1, 2, 3]`      | `[4, 5, 6]`    | `[]`           |
| `[]`             | `[1, 2]`       | `[]`           |
