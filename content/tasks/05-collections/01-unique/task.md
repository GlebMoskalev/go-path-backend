# Уникальные элементы

Напишите функцию `Unique`, которая принимает `[]int` и возвращает новый слайс, содержащий только уникальные элементы в **порядке первого появления**.

Используйте `map[int]bool` как множество для отслеживания уже встреченных значений.

## Пример

| Вход                  | Выход           |
|-----------------------|-----------------|
| `[1, 2, 2, 3, 1]`    | `[1, 2, 3]`     |
| `[5, 5, 5]`           | `[5]`           |
| `[1, 2, 3]`           | `[1, 2, 3]`     |
| `[]`                  | `[]`            |
