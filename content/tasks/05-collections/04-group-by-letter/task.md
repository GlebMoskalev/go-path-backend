# Группировка по первой букве

Напишите функцию `GroupByFirstLetter`, которая принимает `[]string` и возвращает `map[string][]string`, где ключ — первая буква слова (строка из одного символа), а значение — слайс слов с этой буквы.

- Порядок слов внутри каждой группы должен соответствовать порядку их появления во входном слайсе
- Пустые строки пропускать (у них нет первой буквы)

## Пример

| Вход                              | Выход                                          |
|-----------------------------------|------------------------------------------------|
| `["apple", "ant", "banana", "ax"]`| `{"a": ["apple","ant","ax"], "b": ["banana"]}` |
| `["go", ""]`                      | `{"g": ["go"]}`                                |
| `[]`                              | `{}`                                           |
