# Частота слов

Напишите функцию `WordFrequency`, которая принимает строку и возвращает `map[string]int` — словарь, где ключ — слово, значение — количество его вхождений.

Слова разделяются пробелами. Используйте `strings.Fields` для разбиения на слова (он корректно обрабатывает несколько пробелов подряд и ведущие/завершающие пробелы).

## Пример

| Вход                        | Выход                          |
|-----------------------------|--------------------------------|
| `"go is go"`                | `{"go": 2, "is": 1}`           |
| `"a b c a"`                 | `{"a": 2, "b": 1, "c": 1}`     |
| `""`                        | `{}` (пустой map)              |
| `"hello"`                   | `{"hello": 1}`                 |
