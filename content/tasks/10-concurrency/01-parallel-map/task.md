# Параллельное преобразование

Напишите функцию `ParallelMap`, которая принимает `[]int` и функцию `fn func(int) int`, и применяет `fn` к каждому элементу **параллельно** через горутины.

Требования:
- Запустить горутину для каждого элемента
- Дождаться завершения всех горутин через `sync.WaitGroup`
- Вернуть результирующий слайс той же длины
- Порядок элементов в результате должен совпадать с исходным

## Пример

```go
result := ParallelMap([]int{1, 2, 3, 4}, func(n int) int {
    return n * n
})
// result = [1, 4, 9, 16]
```

| Вход           | `fn`     | Выход           |
|----------------|----------|-----------------|
| `[1, 2, 3]`    | `n * 2`  | `[2, 4, 6]`     |
| `[5]`          | `n + 1`  | `[6]`           |
| `[]`           | любая    | `[]`            |
