# Потокобезопасный счётчик

Наивный счётчик без синхронизации даёт гонку данных при одновременном увеличении из нескольких горутин. Реализуйте `SafeCounter` — потокобезопасную версию через `sync.Mutex`.

Реализуйте структуру `SafeCounter` с методами:
- `Inc()` — увеличивает счётчик на 1 (защищено мьютексом)
- `Value() int` — возвращает текущее значение (защищено мьютексом)

Функция `CountConcurrently(n int) int` должна запустить `n` горутин, каждая из которых вызывает `Inc()` один раз, дождаться их завершения и вернуть итоговое значение счётчика.

## Пример

```go
CountConcurrently(100)   // 100
CountConcurrently(1000)  // 1000
```
