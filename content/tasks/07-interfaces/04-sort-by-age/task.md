# Сортировка по возрасту

Реализуйте тип `ByAge` — обёртку над `[]Person`, которая реализует интерфейс `sort.Interface`. Это позволит сортировать людей по возрасту с помощью стандартной функции `sort.Sort`.

Интерфейс `sort.Interface` требует три метода:
- `Len() int` — количество элементов
- `Less(i, j int) bool` — возвращает `true` если элемент `i` должен стоять перед `j`
- `Swap(i, j int)` — меняет элементы местами

Сортируйте по возрасту **по возрастанию**.

## Пример

```go
people := []Person{
    {"Иван", 30},
    {"Анна", 25},
    {"Пётр", 35},
}
sort.Sort(ByAge(people))
// people[0] = {Анна, 25}
// people[1] = {Иван, 30}
// people[2] = {Пётр, 35}
```
