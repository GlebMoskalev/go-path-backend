# Реализация Stringer

Реализуйте метод `String() string` для структуры `Product`, чтобы она удовлетворяла интерфейсу `fmt.Stringer`.

Когда тип реализует `Stringer`, `fmt.Println` и другие функции пакета `fmt` автоматически используют `String()` для его вывода — это и есть неявная реализация интерфейсов в Go.

Метод должен возвращать строку вида: `"Product(name=..., price=...₽)"`.

## Пример

| Name       | Price  | `String()`                              |
|------------|--------|-----------------------------------------|
| `"Ноутбук"`| `59999`| `"Product(name=Ноутбук, price=59999₽)"` |
| `"Книга"`  | `500`  | `"Product(name=Книга, price=500₽)"`     |
