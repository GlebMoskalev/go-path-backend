# Описание типа

Напишите функцию `Describe`, которая принимает `any` и возвращает строку, описывающую тип и значение аргумента.

Используйте `type switch` — идиоматичный Go-способ работать с несколькими возможными типами.

Правила:
- `int` → `"int: <число>"`
- `string` → `"string: <строка>"`
- `bool` и `true` → `"bool: true"`, `false` → `"bool: false"`
- `[]int` → `"[]int с <N> элементами"`
- любой другой тип → `"неизвестный тип"`

## Пример

| Вход            | Выход                   |
|-----------------|-------------------------|
| `42`            | `"int: 42"`             |
| `"hello"`       | `"string: hello"`       |
| `true`          | `"bool: true"`          |
| `[]int{1,2,3}`  | `"[]int с 3 элементами"`|
| `3.14`          | `"неизвестный тип"`     |
