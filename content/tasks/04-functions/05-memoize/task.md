# Мемоизация Фибоначчи

Наивная рекурсия для Fibonacci(40) делает миллионы вызовов — Go не оптимизирует хвостовую рекурсию, и каждый вызов занимает место в стеке.

Напишите функцию `MakeFibMemo`, которая возвращает функцию, вычисляющую числа Фибоначчи **с кешированием** (мемоизация). Кеш должен храниться внутри замыкания — `map[int]int`.

```go
fib := MakeFibMemo()
fib(10) // 55
fib(10) // 55, берётся из кеша
```

Каждый вызов `MakeFibMemo()` создаёт **новый** независимый кеш.

## Пример

| Вызов    | Выход   |
|----------|---------|
| `fib(0)` | `0`     |
| `fib(1)` | `1`     |
| `fib(10)`| `55`    |
| `fib(20)`| `6765`  |
| `fib(30)`| `832040`|
