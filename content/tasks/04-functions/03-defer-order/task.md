# Порядок defer

Напишите функцию `CollectDeferred`, которая принимает слайс строк `labels []string` и возвращает новый слайс — те же строки, но в **обратном порядке**.

**Ограничение:** для разворота используйте только механизм `defer`. Добавляйте каждый элемент к результату через `defer`-вызов, и они выполнятся в LIFO-порядке.

Подсказка: замыкание в `defer` захватывает переменную по ссылке. Чтобы сохранить значение итерационной переменной — создайте локальную копию внутри цикла.

## Пример

| Вход                   | Выход                  |
|------------------------|------------------------|
| `["a", "b", "c"]`      | `["c", "b", "a"]`      |
| `["first", "second"]`  | `["second", "first"]`  |
| `["only"]`             | `["only"]`             |
| `[]`                   | `[]`                   |
