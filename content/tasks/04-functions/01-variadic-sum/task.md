# Сумма чисел

Напишите функцию `Sum`, которая принимает любое количество целых чисел и возвращает их сумму.

Используйте вариадический параметр `...int`. Внутри функции параметр будет слайсом `[]int`.

## Пример

| Вход              | Выход |
|-------------------|-------|
| `Sum(1, 2, 3)`    | `6`   |
| `Sum(10, -5, 3)`  | `8`   |
| `Sum()`           | `0`   |
| `Sum(42)`         | `42`  |

Обратите внимание: `Sum()` без аргументов должна возвращать 0 — это нулевое значение для `int`.
