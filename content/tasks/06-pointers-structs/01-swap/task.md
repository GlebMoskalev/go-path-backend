# Обмен через указатели

Напишите функцию `Swap`, которая принимает два указателя `*int` и меняет значения по адресам местами.

Это демонстрирует разницу между value semantics и pointer semantics: если бы функция принимала `int` напрямую, изменения не отразились бы на вызывающем коде.

## Пример

```go
a, b := 10, 20
Swap(&a, &b)
// a == 20, b == 10
```

| До `Swap`  | После `Swap`  |
|------------|---------------|
| `a=10, b=20` | `a=20, b=10` |
| `a=5, b=5`   | `a=5, b=5`   |
| `a=0, b=1`   | `a=1, b=0`   |
