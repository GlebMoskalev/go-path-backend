# Стек

Реализуйте структуру данных «стек» (LIFO — последним вошёл, первым вышел).

Структура `Stack` уже объявлена в шаблоне. Реализуйте три метода:

- `Push(val int)` — добавляет элемент на вершину стека
- `Pop() (int, bool)` — снимает элемент с вершины и возвращает `(значение, true)`. Если стек пустой — `(0, false)`
- `IsEmpty() bool` — возвращает `true`, если стек пустой

Методы `Push` и `Pop` должны изменять состояние стека, поэтому используйте **pointer receiver**.

## Пример

```go
s := &Stack{}
s.Push(1)
s.Push(2)
s.Pop()    // (2, true)
s.Pop()    // (1, true)
s.Pop()    // (0, false)
s.IsEmpty() // true
```
