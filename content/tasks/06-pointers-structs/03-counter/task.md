# Счётчик

Реализуйте структуру `Counter` с тремя методами:

- `Inc()` — увеличивает счётчик на 1
- `Dec()` — уменьшает счётчик на 1 (минимум 0, не уходит в отрицательные значения)
- `Value() int` — возвращает текущее значение

Все три метода должны использовать **pointer receiver** `*Counter`. Попробуйте вызвать `Inc` через value (не pointer) — Go не сможет модифицировать оригинал.

## Пример

```go
c := &Counter{}
c.Inc()   // 1
c.Inc()   // 2
c.Dec()   // 1
c.Dec()   // 0
c.Dec()   // 0 (не уходит в минус)
c.Value() // 0
```

| Действие | Значение |
|----------|----------|
| Начало   | `0`      |
| `Inc()`  | `1`      |
| `Inc()`  | `2`      |
| `Dec()`  | `1`      |
| `Dec()`  | `0`      |
| `Dec()`  | `0`      |
