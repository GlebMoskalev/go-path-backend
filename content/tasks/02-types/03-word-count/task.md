# Подсчёт слов

Напишите функцию `WordCount`, которая принимает строку и возвращает `map[string]int` — количество вхождений каждого слова. Слова разделены пробелами, регистр не учитывается (приводите к нижнему).

## Пример

| Вход                    | Выход                                    |
|-------------------------|------------------------------------------|
| `"go go go"`            | `{"go": 3}`                              |
| `"Hello hello World"`   | `{"hello": 2, "world": 1}`              |
| `""`                    | `{}`                                     |
