# Сумма элементов слайса

Напишите функцию `Sum`, которая принимает слайс целых чисел и возвращает их сумму.

## Пример

| Вход        | Выход |
|-------------|-------|
| `[1, 2, 3]` | `6`   |
| `[]`        | `0`   |
| `[-1, 1]`   | `0`   |
| `[100]`     | `100` |
