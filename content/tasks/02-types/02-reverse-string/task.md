# Переворот строки

Напишите функцию `Reverse`, которая принимает строку и возвращает её в обратном порядке. Функция должна корректно работать с Unicode-символами.

## Пример

| Вход        | Выход       |
|-------------|-------------|
| `"hello"`   | `"olleh"`   |
| `"Привет"`  | `"тевирП"`  |
| `""`        | `""`        |
| `"a"`       | `"a"`       |
