# Конвертер температур

В коде уже определены именованные типы `Celsius` и `Fahrenheit` на основе `float64`.

Напишите функцию `ToFahrenheit`, которая принимает `Celsius` и возвращает `Fahrenheit`.

Формула: `F = C * 9/5 + 32`

Обратите внимание: `Celsius` и `Fahrenheit` — разные типы, хотя оба основаны на `float64`. Go не позволит смешивать их без явного преобразования. Это и есть суть именованных типов.

## Пример

| Вход (°C) | Выход (°F) |
|-----------|------------|
| `0`       | `32`       |
| `100`     | `212`      |
| `-40`     | `-40`      |
| `37`      | `98.6`     |
