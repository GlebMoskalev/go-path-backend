# Обрезка строки

Напишите функцию `Truncate`, которая принимает строку и максимальную длину (`maxLen int`), и возвращает строку, обрезанную до `maxLen` **символов** (рун).

Если исходная строка длиннее `maxLen`, обрезать до `maxLen-3` символов и добавить `"..."`.
Если строка короче или равна `maxLen` — вернуть без изменений.

Важно: длина считается в **рунах**, не в байтах. Для работы с рунами используйте `[]rune(s)`.

## Пример

| Вход (`s`, `maxLen`)         | Выход                   |
|------------------------------|-------------------------|
| `"Hello, World!"`, `5`       | `"He..."`               |
| `"Hello"`, `10`              | `"Hello"`               |
| `"Привет, мир!"`, `8`        | `"Приве..."`            |
| `"Go"`, `2`                  | `"Go"`                  |
| `"abc"`, `3`                 | `"abc"`                 |

Edge cases:
- Если `maxLen <= 3` — возвращайте первые `maxLen` рун без `"..."`
