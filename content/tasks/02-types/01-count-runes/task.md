# Длина строки в символах

Напишите функцию `RuneCount`, которая принимает строку и возвращает количество **Unicode-символов** (рун) в ней.

В Go `len(s)` возвращает количество **байтов**, а не символов. Например, для строки `"Привет"` `len()` вернёт 12, а не 6 — потому что каждый кириллический символ занимает 2 байта в UTF-8.

## Пример

| Вход       | `len()` | `RuneCount()` |
|------------|---------|---------------|
| `"hello"`  | `5`     | `5`           |
| `"Привет"` | `12`    | `6`           |
| `""`       | `0`     | `0`           |
| `"Go 🚀"`  | `7`     | `4`           |
