# Парсинг цены

Напишите функцию `ParsePrice`, которая принимает строку и пытается распарсить её как число с плавающей точкой.

Функция должна возвращать два значения:
- `float64` — распарсенное число, или `0` если строка некорректна
- `bool` — `true` если парсинг успешен, `false` если нет

## Пример

| Вход       | Выход           |
|------------|-----------------|
| `"19.99"`  | `(19.99, true)` |
| `"100"`    | `(100.0, true)` |
| `"abc"`    | `(0, false)`    |
| `""`       | `(0, false)`    |
| `"1.2.3"`  | `(0, false)`    |
