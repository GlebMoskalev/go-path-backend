# Парсинг цены

Напишите функцию `ParsePrice`, которая принимает строку с ценой (например `"19.99"`) и возвращает два значения: `float64` и `bool`.

- Если строка корректная — возвращает `(значение, true)`
- Если строка некорректная — возвращает `(0, false)`

Используйте `strconv.ParseFloat`. Не обрабатывайте отрицательные цены отдельно — `strconv.ParseFloat` сам справится.

## Пример

| Вход       | Выход           |
|------------|-----------------|
| `"19.99"`  | `(19.99, true)` |
| `"100"`    | `(100.0, true)` |
| `"abc"`    | `(0, false)`    |
| `""`       | `(0, false)`    |
| `"1.2.3"`  | `(0, false)`    |
