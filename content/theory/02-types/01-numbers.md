# Числовые типы

Go предоставляет богатый набор числовых типов с явным указанием разрядности.

## Целые числа

| Тип      | Размер   | Диапазон                       |
|----------|----------|--------------------------------|
| `int8`   | 8 бит    | −128 … 127                     |
| `int16`  | 16 бит   | −32 768 … 32 767               |
| `int32`  | 32 бита  | −2 147 483 648 … 2 147 483 647 |
| `int64`  | 64 бита  | −9.2 × 10¹⁸ … 9.2 × 10¹⁸      |
| `int`    | 32/64    | Зависит от платформы            |

### Беззнаковые

| Тип      | Размер   | Диапазон          |
|----------|----------|--------------------|
| `uint8`  | 8 бит    | 0 … 255            |
| `uint16` | 16 бит   | 0 … 65 535         |
| `uint32` | 32 бита  | 0 … 4 294 967 295  |
| `uint64` | 64 бита  | 0 … 1.8 × 10¹⁹    |
| `byte`   | 8 бит    | Алиас для `uint8`  |

## Числа с плавающей точкой

```go
var pi float64 = 3.141592653589793
var e  float32 = 2.7182818
```

| Тип       | Размер  | Точность        |
|-----------|---------|-----------------|
| `float32` | 32 бита | ~7 знаков       |
| `float64` | 64 бита | ~15 знаков      |

> **Правило:** по умолчанию используйте `float64` — он точнее и является типом по умолчанию для литералов.

## Приведение типов

Go **не** выполняет неявное приведение. Нужно делать это явно:

```go
var i int = 42
var f float64 = float64(i)
var u uint = uint(f)
```

### Потеря данных

```go
var big int64 = 1000
var small int8 = int8(big) // small = -24 (переполнение!)
```

## Арифметические операции

```go
a, b := 17, 5

fmt.Println(a + b)   // 22
fmt.Println(a - b)   // 12
fmt.Println(a * b)   // 85
fmt.Println(a / b)   // 3  (целочисленное деление)
fmt.Println(a % b)   // 2  (остаток)
```

## Пакет math

```go
import "math"

fmt.Println(math.Max(3.14, 2.71))  // 3.14
fmt.Println(math.Sqrt(144))         // 12
fmt.Println(math.Pow(2, 10))        // 1024
```
