# Переменные

Go — статически типизированный язык. Каждая переменная имеет фиксированный тип.

## Объявление через var

```go
var name string = "Go"
var age int     // нулевое значение: 0
```

## Короткая запись :=

```go
lang := "Go"       // тип выводится автоматически — string
year := 2009       // int
```

Короткая запись работает **только внутри функций**.

## Нулевые значения

| Тип       | Нулевое значение |
|-----------|-----------------|
| `int`     | `0`             |
| `float64` | `0.0`           |
| `string`  | `""`            |
| `bool`    | `false`         |
| `pointer` | `nil`           |

## Множественное объявление

```go
var (
    host = "localhost"
    port = 8080
)

a, b, c := 1, "hello", true
```
