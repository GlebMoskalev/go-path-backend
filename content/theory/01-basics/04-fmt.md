# Ввод и вывод

Пакет `fmt` — основной инструмент для работы с текстовым вводом-выводом в Go.

## Вывод

```go
fmt.Print("без переноса")
fmt.Println("с переносом строки")
fmt.Printf("форматированный: %d\n", 42)
```

## Глаголы форматирования

| Глагол | Описание                   | Пример               |
|--------|----------------------------|-----------------------|
| `%d`   | Целое число                | `fmt.Printf("%d", 42)` → `42` |
| `%f`   | Число с плавающей точкой   | `fmt.Printf("%.2f", 3.14)` → `3.14` |
| `%s`   | Строка                     | `fmt.Printf("%s", "Go")` → `Go` |
| `%v`   | Значение в формате по умолчанию | `fmt.Printf("%v", true)` → `true` |
| `%T`   | Тип значения               | `fmt.Printf("%T", 42)` → `int` |
| `%q`   | Строка в кавычках          | `fmt.Printf("%q", "Go")` → `"Go"` |
| `%p`   | Указатель                  | `fmt.Printf("%p", &x)` |
| `%b`   | Двоичное представление     | `fmt.Printf("%b", 10)` → `1010` |

## Sprintf — форматирование в строку

```go
s := fmt.Sprintf("Привет, %s! Тебе %d лет.", "Гофер", 3)
// s = "Привет, Гофер! Тебе 3 лет."
```

## Чтение ввода

```go
var name string
fmt.Print("Как тебя зовут? ")
fmt.Scan(&name)
fmt.Printf("Привет, %s!\n", name)
```

### Чтение нескольких значений

```go
var a, b int
fmt.Scan(&a, &b)        // ввод: 10 20
fmt.Println(a + b)      // 30
```

### Scanf — форматированный ввод

```go
var hour, min int
fmt.Scanf("%d:%d", &hour, &min)  // ввод: 14:30
```

## Вывод в io.Writer

```go
fmt.Fprintln(os.Stderr, "ошибка!")
```

> **Совет:** используйте `%+v` для вывода структур с именами полей.
