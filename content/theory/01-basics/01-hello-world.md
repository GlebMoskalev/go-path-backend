# Hello, World!

Каждая программа на Go начинается с объявления пакета. Исполняемые программы всегда принадлежат пакету `main`.

## Минимальная программа

```go
package main

import "fmt"

func main() {
    fmt.Println("Hello, World!")
}
```

## Разбор

- **`package main`** — указывает, что это исполняемый пакет, а не библиотека.
- **`import "fmt"`** — подключает стандартный пакет форматированного ввода-вывода.
- **`func main()`** — точка входа. Вызывается автоматически при запуске программы.

## Запуск

```bash
go run main.go
```

> **Совет:** используйте `go run .` если в директории несколько `.go`-файлов.
