# Константы

Константы — значения, известные на этапе компиляции. Они не могут быть изменены после объявления.

## Объявление

```go
const pi = 3.14159
const greeting string = "Hello"
```

## Группировка

```go
const (
    StatusOK    = 200
    StatusNotFound = 404
    StatusError = 500
)
```

## iota — автоинкремент

`iota` начинается с `0` и увеличивается на `1` для каждой следующей константы в блоке.

```go
const (
    Sunday = iota  // 0
    Monday         // 1
    Tuesday        // 2
    Wednesday      // 3
    Thursday       // 4
    Friday         // 5
    Saturday       // 6
)
```

## Пропуск значения

```go
const (
    _  = iota       // 0 — пропускаем
    KB = 1 << (10 * iota) // 1 << 10 = 1024
    MB                     // 1 << 20
    GB                     // 1 << 30
)
```

## Типизированные vs нетипизированные

```go
const x = 10         // нетипизированная — подстраивается под контекст
const y int = 10     // типизированная — строго int
```

Нетипизированные константы более гибкие: их можно использовать в выражениях с разными типами без явного приведения.

```go
const ratio = 3.0

var f float32 = ratio  // ОК — ratio адаптируется
var i int = ratio      // ОК — ratio = 3.0, целое значение
```
